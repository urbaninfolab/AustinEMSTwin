Austin EMS Response Time Map Information

About

This page provides interactive EMS response time visualization for Austin, Texas. The map displays average response times by block group, showing spatial patterns in emergency medical service accessibility.

Data Description

The data analyzed includes:

  • Response Time: Time between EMS dispatch and arrival
  • Demographics: Income, poverty, unemployment, racial demographics
  • Infrastructure: Road density, structure age, closest EMS distance
  • Traffic Events: Crash speed, time patterns (daytime/nighttime, weekday/weekend)
  • Environmental: Weather conditions, seasonality

Credits

Created by the Urban Information Lab under the Good Systems Initiative at The University of Texas at Austin.

Additional support for this project comes from the Texas Advanced Computing Center.
